# JSOS: An Operating System in JavaScript

For [CS422](http://labouseur.com/courses/os/) at Marist.

## Loginator

Assumes node.js and npm are installed

1. `cd loginator/`
2. `npm install`
3. `./run`

## Supergrep

1. `cd supergrep/`
2. `npm install`
3. `./run_local`
4. go [here](http://localhost:3000/)
